Smart Canteen Payment System

Team: Yashaj Gupta (EC, 2023-27), Rhythm Agrawal (EC, 2023-27)
Equipment: ESP32 Microcontrollers, QR/ID Scanners, LCD Display, Local SQL/Firebase Database
Project Abstract

The Smart Canteen Payment System is developed to address the challenges faced in canteens where conventional UPI-based transactions often fail due to poor network connectivity. This project establishes a local offline payment ecosystem that eliminates the dependency on external internet signals.

Each student is assigned a unique Student ID card, which acts as a digital wallet linked to their prepaid account. Students can make payments by simply scanning their ID card, after which the transaction is processed and the balance updated in real-time on the server. This approach not only reduces waiting time and eliminates cash handling but also ensures reliable, transparent, and efficient canteen operations even in areas with limited connectivity.

Technologies Used:
ESP32 QR Scanning SQL Database Firebase Local Network

IoT Based Password Door Locking System

Team: Aamodit Saini (EC, 2022-26), Vidhi Awasthi (EC, 2022-26)
Equipment: Arduino Uno, 4x4 Matrix Keypad, SG90 Micro Servo Motor, 16x2 LCD Display, I2C LCD Backpack Module, Buzzer
Project Abstract

In this project, we used an Arduino and a keypad to create a password-based security system. Thefts and frauds are becoming more common by the day, therefore security is becoming a serious worry. As a result, a smart lock with a digital code can simply secure our home, business, locker, and other valuables.

It only unlocks a door when the correct password is supplied. Due to a password-based door lock mechanism, only authorized personnel are permitted access to the restricted areas. The Arduino is in charge of the project's overall operation. A 4×4 keypad can be used to input the necessary password.

Technologies Used:
Arduino Uno Matrix Keypad Servo Motor LCD Display I2C Module
