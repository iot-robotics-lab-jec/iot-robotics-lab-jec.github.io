Development Boards & Microcontrollers

Raspberry Pi 4

A powerful single-board computer with ARM Cortex-A72 processor, ideal for AI, machine learning, and complex IoT applications. Features GPIO pins, multiple USB ports, and wireless connectivity.

Learn More
ESP8266 Development Board

Wi-Fi enabled microcontroller perfect for IoT projects. Low power consumption with built-in wireless capabilities, ideal for home automation and sensor networks.

Learn More
ATmega2560 Development Board

Arduino Mega compatible board with 54 digital I/O pins and 16 analog inputs. Perfect for complex projects requiring multiple sensors and actuators.

STM32 Nucleo Boards

Professional ARM Cortex-M development boards offering high performance and advanced features for embedded systems development and real-time applications.

Learn More
LPC2148 Development Board

ARM7TDMI-S based microcontroller development board with 512KB flash memory. Excellent for learning embedded systems and real-time programming.

P89V51RD2 Development Board

8051 family microcontroller board perfect for learning basic embedded programming concepts and digital electronics fundamentals.

Tiva Launchpad

Texas Instruments ARM Cortex-M4F based development platform with floating-point unit. Ideal for signal processing and control applications.

Learn More
Altera Cyclone IV FPGA, DE10-Nano

Field Programmable Gate Array development kit combining FPGA fabric with ARM Cortex-A9 processor. Perfect for digital signal processing and hardware acceleration projects.

Learn More

Communication Modules

NRF24L01

2.4GHz wireless transceiver module for short-range communication between microcontrollers. Ideal for wireless sensor networks and remote control applications.

Zigbee Modules (100m range)

Low-power mesh networking wireless modules supporting Zigbee protocol. Perfect for building smart home systems and IoT networks with extended range.

Zigbee Module Adapter

Interface adapter boards for easy integration of Zigbee modules with development boards and breadboards for prototyping wireless networks.

Sensors & Input Devices

TCS3200 Based Color Sensor Module

RGB color sensor module capable of detecting and measuring color values. Useful for color sorting, object detection, and automated quality control systems.

GY-87 IMU Module

10-DOF Inertial Measurement Unit combining accelerometer, gyroscope, magnetometer, and barometer. Essential for drone flight control, robotics navigation, and motion tracking.

HD USB WebCam

High-definition USB camera for computer vision projects, surveillance systems, and real-time image processing applications.

Motors & Actuators

Quad Encoder Geared DC Motor (200RPM, 12V DC)

High-precision geared DC motor with quadrature encoder for accurate position feedback. Ideal for robotic wheels, conveyor systems, and precise positioning applications.

Metal-Gear Servo Motors

High-torque servo motors with metal gears for durability and precision. Perfect for robotic arms, steering mechanisms, and heavy-duty positioning systems.

Micro Servo Motor

Compact servo motors for light-duty applications requiring precise angular positioning. Commonly used in small robots, camera gimbals, and model control systems.

Servo Motor Based Gripper Kit

Robotic gripper assembly with servo motor control for pick-and-place operations. Essential component for robotic arm projects and automated manipulation tasks.

Control & Power Systems

Motor Driver Module (8 Amp, 6V-36V)

High-current motor driver capable of controlling powerful DC motors and stepper motors. Features current limiting and thermal protection for safe operation.

2-Channel Relay Board (12 Volt)

Dual relay module for controlling high-voltage AC appliances and heavy-duty DC loads. Essential for home automation and industrial control applications.

2-Channel Relay Board (5 Volt)

Low-voltage relay module compatible with microcontroller logic levels. Perfect for switching smaller loads and interfacing with digital control systems.

Accessories & Storage

Micro SD Card (32 GB)

High-capacity storage for Raspberry Pi operating systems, data logging, and storing project files and sensor data.

USB Adapter 2A

High-current USB power adapter for powering development boards and charging devices. Provides stable power supply for demanding projects.
